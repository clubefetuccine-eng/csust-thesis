## 📘 What This Is

csust-thesis is a LaTeX template for a Master Thesis of CSUST. It helps you build a clean thesis document with a fixed structure, standard sections, and common academic formatting.

Use it if you want a simple way to start your thesis without setting up the layout from scratch.

## 🪟 How to Use on Windows

This template works as a file set for LaTeX, so you do not install it like a normal app. You download it, unpack it, and open the main thesis file in a LaTeX editor.

### Step 1: Download the release
1. Open the releases page.
2. Find the latest version.
3. Download the source package or release archive.

### Step 2: Unpack the files
1. Right-click the downloaded file.
2. Choose Extract All or use your ZIP tool.
3. Save the files in a folder you can find later, such as Documents or Desktop.

### Step 3: Open the thesis project
1. Open the extracted folder.
2. Look for the main `.tex` file.
3. Open it in a LaTeX editor such as TeXstudio or VS Code with LaTeX support.

### Step 4: Build the PDF
1. Use the editor’s build button.
2. Let the tool compile the document.
3. Open the generated PDF when the build finishes.

## ✍️ Basic Editing Steps

### 1. Change the title page
Edit the thesis title, your name, your supervisor, and your school details in the main file or cover page file.

### 2. Write your abstract
Replace the sample abstract with your own thesis summary. Keep it short and clear.

### 3. Add your chapters
Put your thesis content into the chapter files. Each chapter should focus on one part of your work.

### 4. Update references
Add your sources to the reference file and make sure the citation style matches your school rules.

### 5. Insert figures and tables
Add charts, images, and tables where needed. Keep the captions clear and short.

## 🧭 Typical Folder Layout

A thesis template like this often uses a structure such as:

- `main.tex` for the full document
- `chapters/` for chapter content
- `figures/` for images
- `references/` or `bib/` for citations
- `style/` for formatting settings

If your extracted folder uses different names, open the main `.tex` file first. That file usually shows how the project is organized.

## 🔧 Common Tasks

### Change page content
Open the main `.tex` file and edit the text in the same place where the sample content appears.

### Add a new chapter
Copy an existing chapter file, rename it, and add it to the main file if needed.

### Add a new figure
Place the image in the figures folder, then insert it with the image command in your chapter.

### Fix citation issues
Check that each citation key in your text matches a key in the bibliography file.
